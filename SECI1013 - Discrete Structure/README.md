## 🧠 Reflection
In this course, I gained mathematical concepts such as set theory, relations, functions and graph theory. I learned how logic, proofs and combinatorics are applied to algorithms and data structures. This course has trained me to be logical and think in a well-organized manner with improved systems in programming and designing systems as well. From my point of view, including more real-life examples and programming-based exercises could make abstract concepts more relatable and easier to understand. Now, I understand how mathematical logic supports almost every aspect of computing science, from database design to software design, which I not only apply in math problems but in programming as well.
