## 🧠 Reflection
During this course, I gained knowledges about algorithms, flow of control (loops and conditions) and modular programming through functions. I learned how to use programming languages like C++ to solve problems patiently. In my opinion, I believe that mini projects like a simple calculator and a simple game like Tic-Tac-Toe would improve my programming skills and logical thinking.  It always feels great and satisfying that my program was able to run without any errors. At the end of the course, I was able to write programs for data processing, calculations and solving problems. I hope I can improve my logical thinking and problem solving skills throughout this course so I am able to solve more complicated coding problems and challenges.
